# Stack62 — Competitive Landscape

## Competitive Positioning

Stack62 sits at the intersection of three markets that have historically not overlapped: **AI assistants**, **no-code platforms**, and **business ERP**. No competitor currently occupies this position.

---

## Direct Competitors

### No-Code / Low-Code Platforms

| Competitor | Strength | Stack62 Advantage |
|-----------|---------|------------------|
| **Airtable** | Spreadsheet-like familiarity, large ecosystem | Stack62 generates systems from plain language; no manual table design needed. Stack62 systems include workflows, document generation, and AI assistance natively. |
| **Notion** | Docs + databases, widespread adoption | Notion is a document tool; it has no workflow engine, approval chains, or AI system governance. Stack62 is an operating system, not a wiki. |
| **Monday.com** | Project tracking, strong sales | Monday.com requires significant setup and is optimised for project management, not custom business processes. Stack62 auto-builds the schema. |
| **Bubble** | Full web app builder | Bubble requires visual programming — not accessible to non-technical users. Stack62 requires only natural language. |
| **Glide / AppSheet** | Mobile-first app builder from spreadsheets | Requires existing spreadsheet data; no AI-driven system design; no workflow automation at depth. |

### AI Coding / App Generation

| Competitor | Strength | Stack62 Advantage |
|-----------|---------|------------------|
| **Replit** | Code generation, runs in browser | Replit generates and exports code — the app leaves the platform. Stack62 systems stay on the platform with continuous AI assistance, governance, and version control. |
| **Cursor / GitHub Copilot** | Developer productivity | Developer-only. Stack62 targets non-technical business owners and operators. |
| **v0 (Vercel)** | Frontend generation | UI only; no backend, no data, no workflows. Stack62 is a complete operating system. |
| **Bolt.new** | Full-stack app generation | Generated code is exported and self-hosted. No governance, no continuous AI assistance, no multi-tenant infrastructure. |

### Business Operating Systems / ERP

| Competitor | Strength | Stack62 Advantage |
|-----------|---------|------------------|
| **Salesforce** | Market leader CRM, large ecosystem | Starts at $75/user/month; requires implementation partner; overkill for SMBs. Stack62 costs $199/month flat. |
| **HubSpot** | Freemium inbound + CRM | Marketing-first, not a general business OS. No AI system generation. |
| **Zoho One** | Breadth of SMB tools at low price | 40+ pre-built applications; no AI-native system generation; rigid templates. Stack62 generates custom systems to the business's exact workflow. |
| **Odoo** | Open-source ERP, modular | Requires technical setup; significant configuration overhead; no AI-driven customisation. |

### AI Agents / Assistants

| Competitor | Strength | Stack62 Advantage |
|-----------|---------|------------------|
| **Microsoft Copilot for Business** | Deeply integrated with M365 | Works within M365 only; no standalone system building; no workflow automation outside Microsoft tools. |
| **Google Gemini for Workspace** | Integrated with Google Workspace | Same limitation — assistant overlay, not a system builder. No running business systems. |
| **ChatGPT Enterprise** | Broad capability, brand trust | Conversational; does not generate and run persistent business systems. No data records, no workflows, no governance. |

## Defensible Moats

### 1. Platform Lock-in (Switching Cost)
Systems built on Stack62 run on Stack62. When a customer's CRM, approval workflows, and document templates are all Stack62 systems with 12 months of data and history, the switching cost is high — comparable to switching an ERP, not a SaaS tool.

### 2. AI Feedback Loop (Data Flywheel)
Every Plan → Diff → Approve interaction trains Stack62's understanding of what changes businesses accept vs. reject. Over time, the Coworker's suggestions become more accurate for each industry vertical — a proprietary fine-tuning signal that competitors cannot easily replicate.

### 3. Governance Model (Trust Moat)
The diff-and-approve model is patentable and cannot be copied without changing the fundamental product experience. Competitors who give AI full autonomy over business data will generate customer anxiety; Stack62's governed model wins in regulated and conservative segments.

### 4. Multi-Provider AI (Resilience Moat)
By supporting Anthropic, OpenRouter, and Ollama, Stack62 is not dependent on any single AI provider's pricing or availability. This is a unique capability in the SMB tool space.

### 5. WhatsApp + Paystack (Geographic Moat)
First-mover advantage in African and MENA markets through native Paystack and WhatsApp Business API integration — channels that Western SaaS competitors have not prioritised.

## Competitive Summary

Stack62 has no direct competitor that combines:
- AI-native system generation from natural language
- Running platform (systems stay live, not exported)
- Governance and approval layer
- Multi-channel integrations (Slack, WhatsApp, email)
- Document generation
- Multi-tenant, role-based access control
- Self-hostable AI option

The closest analogies are Replit (for developers) + Airtable (for business data) + Microsoft Power Platform (for automation) — but packaged for non-technical SMB operators at a fraction of the cost.
